❌ PROBLÈMES COURANTS
════════════════════════════════════════════════════════════════════════════════

Problème: "fatal: repository or object not found"
→ Le dépôt n'existe pas sur GitHub. Créez-le d'abord!

Problème: "fatal: Authentication failed"
→ Le token est incorrect ou expiré. Générez-en un nouveau!

Problème: "fatal: branch rename failed"
→ Votre branche s'appelle "master" au lieu de "main"
→ Solution: git branch -M main

Problème: "Everything up-to-date"
→ C'est normal! Les fichiers sont déjà poussés.
→ Pour mettre à jour: git add . && git commit -m "Updates" && git push

Problème: Git n'est pas reconnu
→ Installez Git: https://git-scm.com/download/win
