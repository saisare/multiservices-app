🆘 PROBLÈMES COURANTS
─────────────────────────────────────────────────────────────────────────────

Port utilisé?
  taskkill /F /IM node.exe
  OU trouver le PID spécifique avec: netstat -ano | findstr :3001

MySQL non actif?
  net start MySQL80

npm install échoue?
  rm -r node_modules package-lock.json
  npm install --force

Frontend timeout?
  set NEXT_DISABLE_TURBOPACK=1
  npm run dev

Services ne démarrent pas?
  Regardez le message d'erreur dans le terminal
  Vérifiez le fichier .env du service
